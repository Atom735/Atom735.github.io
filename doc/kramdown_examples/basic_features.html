Основные возможности Kramdown

Заголовки с атрибутами

Заголовок 1 {#h1-id .h1-class}

Заголовок 2 {#h2-id .h2-class}

Заголовок 3 {#h3-id .h3-class}

Заголовок 4 {#h4-id .h4-class}

Текстовое форматирование

Жирный текст {.bold} Курсивный текст {.italic} Жирный курсив {.bold-italic} ~~Зачеркнутый текст~~ {.strikethrough} `Моноширинный код` {.code}

Ссылки с атрибутами

Обычная ссылка {.link} Ссылка с ID {.internal-link} Ссылка с title {.tooltip}

Изображения

!Alt текст {.responsive}

Списки с атрибутами

Маркированный список

  • Элемент 1
  • Элемент 2
  • Элемент 3
{:.my-list}

Нумерованный список

1. Первый элемент 2. Второй элемент 3. Третий элемент {:.numbered-list}

Список определений

Apple : Плод яблони : Компания из Купертино

Orange : Плод апельсинового дерева : Цвет между красным и желтым

Цитаты

> Это обычная цитата > с несколькими строками > и красивым оформлением {:.quote}

> Это цитата с атрибутами > и специальным стилем {:.highlight-quote}
