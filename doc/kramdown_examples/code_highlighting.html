```python def fibonacci(n): """Вычисляет n-е число Фибоначчи""" if n <= 1: return n return fibonacci(n-1) + fibonacci(n-2)

for i in range(10): print(f"F({i}) = {fibonacci(i)}") ```

```javascript class Calculator { constructor() { this.history = []; }

add(a, b) { const result = a + b; this.history.push(`${a} + ${b} = ${result}`); return result; }

getHistory() { return this.history; } }

const calc = new Calculator(); console.log(calc.add(5, 3)); // 8 ```

Код с номерами строк

```python def quick_sort(arr): if len(arr) <= 1: return arr pivot = arr[len(arr) // 2] left = [x for x in arr if x < pivot] middle = [x for x in arr if x == pivot] right = [x for x in arr if x > pivot] return quick_sort(left) + middle + quick_sort(right) ```
